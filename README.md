# Dashboard Template

A responsive dashboard web application built using HTML and CSS.

## 🚀 Features
- Multi-page dashboard layout
- Responsive design for all screen sizes
- Clean and modern UI
- Organized structure for scalability

## 📂 Pages Included
- Dashboard (Home)
- Courses
- Files
- Friends
- Plans
- Profile
- Projects
- Settings

## 🛠️ Technologies Used
- HTML5
- CSS3
